# 使用最新 LingPal 源码的指南

> [!NOTE] 这是一个 Tauri 项目，建议你先能跑 Frontend_vue 后再来搞这个，此外注意 Rust 的项目编译后非常大（目前已经有 5GB 左右）注意内存。

1. 下载 Rust [安装地址](https://www.rust-lang.org/zh-CN/tools/install)
2. 下载 Node.js [安装地址](https://nodejs.org/en/download)
3. 在 LingChat_Pal 中，在 cmd 窗口输入 pnpm install
4. 启动`start.bat`
